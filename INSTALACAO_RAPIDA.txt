────────────────────────────────────────────────────────────────────

📱 PASSO 1: INSTALAR O APP

   Opção A - Via ADB:
   
   adb install -r app/build/outputs/apk/debug/app-debug.apk
   
   Opção B - Transferir APK para celular e instalar manualmente

────────────────────────────────────────────────────────────────────

🔵 PASSO 2: PAREAR A IMPRESSORA RP4

   1. Ligue a impressora RP4
   2. No celular: Configurações → Bluetooth
   3. Procure por "RP4-XXXX" 
   4. Toque para parear
   5. Aguarde "Conectado"

────────────────────────────────────────────────────────────────────

🎨 PASSO 3: USAR O APP

   1. Abra o app "RP4 Printer"
   2. Toque em "Conectar à Impressora"
   3. Selecione sua RP4 da lista
   4. Aguarde "✓ Conectado!"
   5. Desenhe uma assinatura na área branca
   6. Toque em "Imprimir"
   7. ✅ Assinatura deve sair como IMAGEM!

────────────────────────────────────────────────────────────────────

🔧 O QUE FOI CORRIGIDO:

   ❌ ANTES: Imprimia caracteres especiais
   ✅ AGORA: Imprime imagem corretamente!
   
   🔹 Mudou de comando GS v 0 para ESC * (linha por linha)
   🔹 Mais compatível com RP4
   🔹 Não depende mais do SDK Honeywell

────────────────────────────────────────────────────────────────────

⚠️  SE AINDA IMPRIMIR CARACTERES ESPECIAIS:

   1. Limpe o cache do app:
      
      adb shell pm clear com.honeywell.rp4printer
      
   2. Desinstale a versão antiga:
      
      adb uninstall com.honeywell.rp4printer
      
   3. Instale novamente:
      
      adb install app/build/outputs/apk/debug/app-debug.apk
      
   4. Reinicie a impressora RP4

────────────────────────────────────────────────────────────────────

🐛 TROUBLESHOOTING:

   Problema: Não conecta
   Solução: 
   - Verifique se está pareado
   - Distância < 3 metros
   - Reinicie a RP4
   
   Problema: Permissões negadas
   Solução:
   - Configurações → Apps → RP4 Printer → Permissões
   - Conceda Bluetooth e Localização (Android < 12)
   
   Problema: Imagem cortada
   Solução:
   - Reduza MAX_IMAGE_WIDTH para 256
   - Aumente delays para 500ms
